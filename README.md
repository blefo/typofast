# Typofast

Typofast suggests the next words as you type on your Mac. It uses a local on-device LLM to predict what comes next—no account, no cloud typing. OCR and context from the app you’re in help completions stay accurate to what you’re actually writing.

Open Typofast from the menu bar, allow the permissions it asks for, and start typing in any app. Accept a suggestion with Tab (or dismiss it and keep typing).

## Requirements

- macOS
- Apple Silicon Mac recommended

## Contributions

Contributions are very welcome. Helpful directions include:

- **Improved caching** — faster, smarter reuse of context between suggestions
- **Improved UI** — clearer settings, overlays, and menu bar experience

Open an issue or pull request if you want to help.
